How to Open a .JAR file

Matthew Brompton

Image from [Wikimedia].

Step 1

First check if you have Java installed

  • Open a command line
  • type ‘java -version’ (without quotes)

If Java is installed, go to step 3:

Java is NOT installed,go to step 2:



Step 2

If Java is not installed: Install Java from Oracle.

Step 3

Next, double click your .jar file. If this does not work you can try Step 4.

Step 4

To run from the command line, open your command line, navigate to the folder containing your .jar file, type ‘java -jar { filename }’.


That should have worked. If it does not work for you, please let me know so I can update this page.
