Free MS Office alternatives

Matthew Brompton

What would any business be without documents, slideshows and spreadsheets? Well probably a lot more fun, but also, a lot less organized. With Microsoft’s choice coming in at £119 (or a costly subscription fee) switching to a free alternative could save some serious cash. Just click the product title, and follow each websites guidance for use and installation.

Libre Office

The Good

  • Works on Linux, Windows and MacOS
  • Has a reader version for Android
    • (Editing in the Android version can be enabled in the settings. This feature is still in it’s experimental faze and by the developer’s own admission, should not be used for mission critical tasks.)
  • Open-source so totally free.
  • Feature rich
  • Exports to most possible formats
  • Extensive Integration

The Bad

  • Dated interfaces
  • Can have difficulty displaying MS Office files.
  • No cloud resources
  • Won’t find some more obscure windows features (for example reference builder)

Great for

  • People who often work alone
  • Larger writing projects
  • Offline working
  • GDPR compliance (since it’s totally offline)

Google Docs

The Good

  • Freeware
  • Usable on most devices
  • Cloud based storage
  • Visually attractive
  • Usable on mobile
  • Excellent Features
  • Great collaboration Tools
  • Integration with other Google Services
  • Exports to multiple formats.
  • Great with Microsoft Office imports

The Bad

  • Online only
    • (Out of the Box, Google Chrome + Plugin required for offline work)
  • Browser based
  • Freeware – paid for by your data.
  • Not GDPR compliant (out of the box).

Great for

  • All your presentations and admin that does not use personal data
  • A Middleman between Microsoft Office and Open Source alternatives
  • Collaboration
  • Accessing documents anywhere

WPS Office Free

The Good

  • Best compatibility with Microsoft Office Documents
  • Versions for Linux, Windows, Android and iOS.
  • Great offline performance
  • Similar interface to Microsoft – Customizable too.
  • 1GB Cloud Storage
  • Adware – free with Adverts
  • Rich feature set

    The Bad

  • Adware – must watch short advert to use some features
  • No real time collaboration
  • No MacOS version (except browser based)
  • Only integration with WPS apps

    Great for

  • People who already use Microsoft office
  • People who need some collaboration (but not much)
  • Offline editing
  • When using LibreOffice and Google Docs becomes clunky, but some Libre features and Collaboration features are not needed.
